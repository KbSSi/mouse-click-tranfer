⚙️ 插件设置

配置插件：
  1. 点击浏览器工具栏中的插件图标
  2. 在弹窗中切换到"设置"标签页
  3. 配置时间戳转换和翻译 API 地址
  4. 保存设置

默认情况下，插件使用内置的转换和模拟翻译功能。

你可以配置自己的 API 来获得更好的翻译效果。

🔧 故障排除

如果插件不工作，请检查：

  • 插件是否已正确安装并启用
  • 是否在 chrome://extensions/ 页面开启了"开发者模式"
  • 页面是否已刷新（安装插件后需要刷新页面）
  • 浏览器控制台是否有错误信息
